# Project 10 WTWR

## Project Description

This react project is designed to recommend clothing for certain weather conditions that are gathered using an opensource weather API. The app request location, temperature, and conditions from the api and compiles a list of clothing articles
that are recommended to be worn. If the conditions are cold A jacket would be recommended to be worn, ect.
